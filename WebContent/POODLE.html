  1. POODLE is Main in The Middle Attack
  2. Its a security vulnerability which forces down grade of negotiated protocol fron TLS to SSLV3
  3. Once Protocol downgrade is successful, attacker can exploit SSL V3 vulnerability like session decryption
  4. It is done by interrupting handshake between Client and Server , resulting in the retry of handshake with next level (weaker) protocol
  5. Prevention: Disable Weaker Protocol from Client as well as Server.
